# plume-demo-web

plume-demo-web 是 [plume](https://github.com/plumephp/plume) 项目的脚手架，它提供了plume默认的最佳实践，用来快速构建你的应用。

我们的目标是通过原生PHP构建一个轻巧、易用、可扩展、高性能的Web开发框架。

详细文档请查看[docs](https://github.com/plumephp/docs)

## 功能特点

- 纯PHP
- 多环境定义
- MVC模式和可定制视图
- 快速实现基于JSON格式的API功能
- 简单易用的API
- 请求和数据库级缓存支持
- 全局异常处理
- 日志
- 性能日志
- 单元测试
- 可定制化
- MVC基本类抽象实现
- 辅助工具类
